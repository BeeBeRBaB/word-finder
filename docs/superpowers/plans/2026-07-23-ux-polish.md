# UX Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Four independent UX fixes from the audit — immediate found/miss feedback, guarded New-theme + dismissable win, localStorage persistence, and accessibility basics — client-side only, no new deps, no build step.

**Architecture:** Fix 1 first because it simplifies the `found` entry shape (drops `glow`/`timer`/`tapped` down to just `{sel}`), which fix 3 then persists. Fixes 2 and 4 are independent. Persistence rides the seeded RNG from spec 1: store `{seed, themeIdx, found}` and regenerate the grid from the seed.

**Tech Stack:** Vanilla ES modules, CSS, `localStorage`, Playwright, `node:test`.

## Global Constraints

- No build step; served verbatim from GitHub Pages; deploy is `git push` to `main` (no branches).
- No new dependencies. `src/rng.js`, `src/puzzle.js`, `src/layout.js` stay PURE (no DOM).
- `computeLayout`/grid geometry unchanged (spec 2 is done — don't touch layout).
- `npm test` + `npm run typecheck` + live suite stay green after each task.
- Every `localStorage` access wrapped in try/catch — storage failure degrades to "no persistence", never throws.

## Task 3: Persist the game across reloads

**Files:** `src/storage.js` (new), `src/main.js`, `tests/unit/storage.test.js` (new), `tests/e2e/ux.spec.js`

**Interfaces:**
- Produces: `save({seed, themeIdx, found})`, `load()` → the object or `null`, `clear()` — all safe against a throwing `localStorage`.

- [ ] **Step 1: Unit test for storage (`tests/unit/storage.test.js`)**

`storage.js` must be testable without a real `localStorage`. Design it to accept an injected store (defaulting to `globalThis.localStorage`). Test round-trip and that a throwing store degrades to `null`/no-throw:

```js
import test from 'node:test';
import assert from 'node:assert/strict';
import { makeStorage } from '../../src/storage.js';

function memStore() {
  const m = new Map();
  return { getItem: k => m.has(k) ? m.get(k) : null, setItem: (k, v) => m.set(k, v), removeItem: k => m.delete(k) };
}

test('save/load round-trips', () => {
  const s = makeStorage(memStore());
  const data = { seed: 42, themeIdx: 3, found: [{ word: 'BEACH', x0: 0, y0: 0, x1: 4, y1: 0 }] };
  s.save(data);
  assert.deepEqual(s.load(), data);
});

test('load returns null when empty', () => {
  assert.equal(makeStorage(memStore()).load(), null);
});

test('a throwing store degrades to null / no throw', () => {
  const bad = { getItem() { throw new Error('nope'); }, setItem() { throw new Error('nope'); }, removeItem() {} };
  const s = makeStorage(bad);
  assert.doesNotThrow(() => s.save({ seed: 1, themeIdx: 0, found: [] }));
  assert.equal(s.load(), null);
});
```

- [ ] **Step 3: Write `src/storage.js`**

```js
const KEY = 'wordfinder-save-v1';

/**
 * @typedef {{seed:number, themeIdx:number, found:{word:string,x0:number,y0:number,x1:number,y1:number}[]}} SaveData
 */

/** @param {Pick<Storage,'getItem'|'setItem'|'removeItem'>} [store] */
export function makeStorage(store = /** @type {any} */ (globalThis).localStorage) {
  return {
    /** @param {SaveData} data */
    save(data) { try { store.setItem(KEY, JSON.stringify(data)); } catch { /* no persistence */ } },
    /** @returns {SaveData|null} */
    load() { try { const s = store.getItem(KEY); return s ? JSON.parse(s) : null; } catch { return null; } },
    clear() { try { store.removeItem(KEY); } catch { /* ignore */ } },
  };
}
```

- [ ] **Step 5: `src/main.js` — capture seed, save on change, restore on boot**

- Import: `import { makeStorage } from './storage.js';` then `const store = makeStorage();`.
- Refactor so each puzzle is built from a FRESH rng seeded by a captured seed (today `newTheme` reuses the advanced `rng`, which can't be reproduced). Track `let currentSeed`. Change `newPuzzle(idx)` to `newPuzzle(seed, idx)`:

```js
function newPuzzle(seed, idx) {
  currentSeed = seed; themeIdx = idx;
  const rng = makeRng(seed);
  state.found = {}; state.foundOrder = []; state.sel = null; state.drag = null; state.miss = null;
  state.puzzle = buildPuzzle({ themes: THEMES, themeIdx: idx, rng, size: N, count: COUNT });
  els.theme.textContent = cap(state.puzzle.name);
  if (state.winTimer) { clearTimeout(state.winTimer); state.winTimer = null; }
  els.win.style.display = 'none';
  layout(); list(); persist();
}
function persist() {
  store.save({ seed: currentSeed, themeIdx,
    found: state.foundOrder.map(w => ({ word: w, ...state.found[w].sel })) });
}
```

- `newTheme` picks a fresh seed + a different theme: `newPuzzle(Date.now() >>> 0, i)`.
- After a find in `endDrag`, call `persist()` (add it right after `list();` in the hit branch).
- Boot logic — replace the bottom `newPuzzle(themeIdx)` call:

```js
const urlSeed = new URLSearchParams(location.search).get('seed');
const urlTheme = new URLSearchParams(location.search).get('theme');
const saved = store.load();
if (urlSeed !== null || urlTheme !== null) {
  newPuzzle(resolveSeed(location.search), resolveThemeIndex(location.search, THEMES.length, makeRng(resolveSeed(location.search))));
} else if (saved) {
  restore(saved);
} else {
  newPuzzle(Date.now() >>> 0, Math.floor(Math.random() * THEMES.length));
}

/** @param {import('./storage.js').SaveData} saved */
function restore(saved) {
  newPuzzle(saved.seed, saved.themeIdx);   // regenerates identical grid, empty found
  for (const f of saved.found) {
    if (!state.puzzle.words.includes(f.word) || state.found[f.word]) continue; // guard stale saves
    state.found[f.word] = { sel: { x0: f.x0, y0: f.y0, x1: f.x1, y1: f.y1 } };
    state.foundOrder.push(f.word);
  }
  pills(); list();   // redraw pills + cross out; do NOT pop the win overlay on restore
}
```

Remove the now-unused top-level `const rng = …` / `let themeIdx = …` seed lines (fold into the boot logic; keep `let themeIdx;` and `let currentSeed;` declared).
